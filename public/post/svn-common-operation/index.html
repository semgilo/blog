mac下svn常用操作命令

svn配置BeyondCompare对比工具

Beyond Compare 官方说明

常规操作

检出工程

svn co url

查看当前状态

svn st

添加文件或者文件夹

svn add . --force

查看当前修改

svn diff

其它

svn help

进阶操作

删除所有不存在的文件和文件夹

svn st|awk '{if($1=="!"){print $2}}'|xargs svn rm

增加？号文件

svn st|awk '{if($1=="?"){print $2}}'|xargs svn add

提交m文件文件

svn st|awk '{if($1=="m"){print $2}}'|xargs svn ci -m "comments"
